row nowrap + flex-start
Os itens são dispostos em uma linha única, alinhados ao início do contêiner.
row wrap + center
Os itens são dispostos em várias linhas, centralizados no contêiner.
row wrap-reverse + space-between
Os itens são dispostos em várias linhas, quebrando conforme necessário, com espaço igual entre eles.
column nowrap + flex-end
Os itens são dispostos em uma coluna única, alinhados ao final do contêiner.
column wrap + space-around
Os itens são dispostos em várias colunas, quebrando conforme necessário, com espaço igual ao redor de cada item.
column wrap-reverse + space-evenly
Os itens são dispostos em várias colunas, quebrando conforme necessário, com espaço igual ao redor de cada item, incluindo as extremidades.
